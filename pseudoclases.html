Pseudoclases relacionadas con hijos de un elemento

Este tipo de pseudoclases permite:

  • Alternar estilos (nth-child(even), nth-child(odd)).
  • Aplicar estilos a elementos únicos (only-child, only-of-type).
  • Crear selecciones más específicas en una estructura HTML compleja
    (first-of-type, last-of-type, nth-of-type).

Pseudoclases básicas

  • :first-child: Selecciona el primer hijo de su padre.
  • :last-child: Selecciona el último hijo de su padre.
  • :nth-child(n): Selecciona el n-ésimo hijo
    div:nth-child(3) { border: 1px solid red; }
    Selecciona el selecciona el tercer hijo de los div y le aplica un borde rojo.
  • :nth-child(even) : Selecciona los hijos en posiciones pares(2, 4, 6, ...).
  • :nth-child(odd) : Selecciona los hijos en posiciones impares (1, 3, 5, ...).

    Esta pseudoclase junto con la anterior, permite lograr el efecto pijama en tablas.

    Ejemplo:
    tr:nth-child(even) {
      background-color: gainsboro; /* Color gris claro para filas pares */
    }

    tr:nth-child(odd) {
      background-color: white; /* Color blanco para filas impares */
    }

Pseudoclases más avanzadas

  • :nth-last-child(n): Igual que :nth-child(n), pero contando desde el final. Ejemplo:
    :nth-last-child(2) { color: red; }
    Selecciona el segundo hijo contando desde el final.
  • :only-child: Selecciona el hijo único dentro de su padre. Ejemplo:
    div:only-child { font-weight: bold; }
    Esta regla se aplica solo si el div no tiene hermanos.
  • :first-of-type: Selecciona el primer hijo de un tipo específico dentro de su padre. Ejemplo:
    article p:first-of-type { color: blue; }
    Selecciona el primer <p> dentro de <article>, sin importar otros elementos.
  • :last-of-type: Selecciona el último hijo de un tipo específico dentro de su padre. Ejemplo:
    article p:last-of-type { color: green; }
    Selecciona el primer <p> dentro de <article>, sin importar otros elementos.
  • :nth-of-type(n): Selecciona el n-ésimo hijo de un tipo específico dentro de su padre. Ejemplo:
    div p:nth-of-type(3) { font-size: 20px; }
    Selecciona el tercer <p> dentro de <div>, sin importar otros tipos de hijos.
  • :nth-last-of-type(n) : Igual que :nth-of-type(n), pero contando desde el final. Ejemplo:
    div p:nth-last-of-type(2) { color: orange; }
    Selecciona el segundo <p> desde el final dentro de <div>.
  • :only-of-type: Selecciona si es el único elemento de su tipo dentro de su padre.Ejemplo:
    section h1:only-of-type { text-transform: uppercase; }
    Se aplica solo si es el único <h1> dentro de <section>.
